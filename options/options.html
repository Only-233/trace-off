止痕 Trace Off

版本 1.0.1

一款轻量级浏览器扩展，让你可以按域名禁止浏览器保存历史记录。

功能特性

  • ✓ 支持顶级域名、多级子域名、localhost 及端口
  • ✓ 弹窗一键快捷添加，域名可编辑
  • ✓ 域名独立开关 + 全局总开关
  • ✓ 全选 / 批量启用 / 禁用 / 删除
  • ✓ 按域名清空历史记录（多选 + 确认）
  • ✓ 全部数据本地存储，隐私安全
